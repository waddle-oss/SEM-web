## 1. 系统概述

### 1.1 项目背景

本系统面向生物学科实验场景，提供一站式的显微图像颗粒分析解决方案。基于深度学习实例分割技术，自动识别并统计SEM（扫描电子显微镜）图像中的颗粒数量、粒径分布及形态参数。

### 1.2 核心功能

| 功能模块 | 描述 |
|----------|------|
| **图像上传** | 支持PNG、JPG、TIF、BMP格式，拖拽或点击上传 |
| **智能分析** | YOLOv8n-seg实例分割，自动识别颗粒轮廓 |
| **比例尺提取** | OCR自动识别图像中的比例尺信息 |
| **粒径统计** | 计算平均直径、D50、方差、最大/最小值 |
| **结果可视化** | 标注图像、粒子表格、统计图表 |
| **用户管理** | 邀请码注册、角色权限控制 |

### 1.3 应用场景

- 生物细胞计数
- 微球颗粒分析
- 材料科学粒径分布统计
- 医药颗粒质量控制

### 4.2 图像分析模块 (core/analyzer.py)

#### 4.2.1 模型安全加载机制

采用"阅后即焚"模式保护模型权重：

```
流程：
1. 读取加密模型文件 (best.encrypted)
2. 使用 AES 解密得到原始字节
3. 写入系统临时文件 (tempfile.NamedTemporaryFile)
4. YOLO 从临时文件加载模型
5. 加载完成后立即删除临时文件
```

#### 4.2.2 分析主流程

```python
def analyze_particles(image_bytes, scale_ratio=None, use_mock=False):
    """
    颗粒分析主函数
    
    步骤：
    1. cv2.imdecode 解码图像
    2. 若未提供 scale_ratio，自动 OCR 提取
    3. YOLOv8 实例分割获取 masks
    4. 循环处理每个 mask:
       - 计算实际面积 (cv2.contourArea)
       - 计算质心 (cv2.moments)
       - 计算完整度 = 实际面积 / 外接圆面积
       - 边缘过滤：排除触碰图像边界的颗粒
       - 完整度过滤：排除 completeness < 0.8 的颗粒
    5. 统计计算：平均直径、D50、方差、最值
    6. 生成标注图像并返回
    """
```

#### 4.2.3 过滤规则

| 规则 | 条件 | 说明 |
|------|------|------|
| **边缘过滤** | 轮廓距边界 ≤ 5像素 | 排除不完整的颗粒 |
| **完整度过滤** | completeness ≥ 0.8 | 排除被截断的颗粒 |
| **面积过滤** | contourArea > 100 像素 | 排除过小的噪点 |

#### 4.2.4 返回数据结构

```python
{
    "total_count": int,           # 有效颗粒总数
    "average_nm": float,          # 平均直径(nm)

    "std_dev": float,            # 标准差(nm)
    "min_nm": float,             # 最小直径(nm)

    "particle_data": [            # 每个颗粒的详细数据
        {
            "id": int,            # 颗粒编号
            "x": float,           # 质心X坐标
            "y": float,           # 质心Y坐标
```

### 4.3 比例尺提取模块 (core/scale_reader.py)

#### 4.3.1 比例尺提取流程

```
┌─────────────────────────────────────────────────────────────┐
│                    输入: SEM 原始图像                         │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│  Step 1: 比例尺区域检测（禁止整图 OCR）                         │
│  优先底部 20% → 底部 30%/40% → 底左/底右角落                    │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│  Step 2: OCR 仅识别标尺文字                                   │
│  支持: "100 nm", "200 nm", "1 μm", "500nm" 等                 │
│  不直接输出 nm/px                                            │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│  Step 3: 图像处理测量标尺线像素长度                             │
│  • 二值化 + 形态学找水平线段                                   │
│  • 行扫描最长连续亮/暗段精修                                   │
│  • 禁止按图像宽度比例估算伪长度                                 │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│  Step 4: 计算比例尺                                           │
│  nm/px = physical_length_nm / pixel_length                   │
│  例: 100 nm / 1020 px = 0.098039 nm/px                       │
└─────────────────────────────────────────────────────────────┘
```

返回字段（节选）：`scale_ratio`, `physical_length_nm`, `pixel_length`, `ocr_text`,
`ocr_raw_text`, `region`, `bar`, 以及可选 `debug`（区域尝试过程、OCR原文、线段长度）。

### 4.4 安全模块 (core/security.py)

#### 4.4.1 加密机制

| 项目 | 说明 |
|------|------|
| **算法** | AES-128-CBC (Fernet) |
| **密钥长度** | 128位 |
| **密钥格式** | Base64编码的32字节 |
| **数据包装** | Base64二次编码 |

#### 4.4.2 密钥管理

```python
SECRET_KEY = b'LrVHm8k9_QmXZdD9Kp3nY2xJw5vE7cR4sUo0bFtG8yI='
_fernet = Fernet(SECRET_KEY)

def encrypt_data(data: bytes) -> str:
    """加密数据，返回Base64编码字符串"""
    encrypted_bytes = _fernet.encrypt(data)
    return base64.b64encode(encrypted_bytes).decode("utf-8")

def decrypt_data(encrypted_str: str) -> bytes:
    """解密数据，返回原始字节"""
    decoded_bytes = base64.b64decode(encrypted_str)
    return _fernet.decrypt(decoded_bytes)
```

```sql
CREATE TABLE `users` (
  `id` INT(11) NOT NULL AUTO_INCREMENT COMMENT '用户ID',

  `password` VARCHAR(255) NOT NULL COMMENT '密码（明文存储，建议优化）',

  `created_at` TIMESTAMP NULL DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  PRIMARY KEY (`id`),
  UNIQUE KEY `username` (`username`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 主键，自增 |
| username | VARCHAR(50) | 用户名，唯一 |
| password | VARCHAR(255) | 密码（当前明文） |
| role | VARCHAR(20) | user/admin |
| created_at | TIMESTAMP | 注册时间 |

#### 6.2.2 邀请码表 (invite_codes)

```sql
CREATE TABLE `invite_codes` (

  PRIMARY KEY (`id`),
  UNIQUE KEY `code` (`code`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 主键，自增 |
| code | VARCHAR(50) | 邀请码，唯一 |
| used_by | VARCHAR(50) | 使用者（NULL=未使用） |

## 7. 安全机制

### 7.1 数据传输加密

```
┌─────────────────────────────────────────────────────────────────────┐
│                          加密传输流程                                │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│   前端                         后端                                  │
│   ─────                        ──────                                │
│                                                                      │
│   image_bytes ───► AES加密 ───► encrypted ───► AES解密 ───► 处理      │
│   ◄───────────────────────── encrypted_result ◄──────────────────    │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### 7.2 模型权重保护

| 保护措施 | 说明 |
|----------|------|
| **磁盘加密** | 模型权重以加密格式 (best.encrypted) 存储 |
| **阅后即焚** | 运行时解密到临时文件，加载后立即删除 |
| **内存加载** | 模型不持久化到磁盘 |

### 7.3 用户认证

| 安全措施 | 说明 |
|----------|------|
| **邀请码注册** | 新用户必须通过邀请码注册 |
| **邀请码限制** | 每个邀请码只能使用一次 |
| **角色权限** | admin 角色才能管理用户和邀请码 |

```json
{
```

```json
{
    "success": true,

}
```

**POST** `/api/analyze_secure`

**请求：**
- Content-Type: `multipart/form-data`
- 字段：
