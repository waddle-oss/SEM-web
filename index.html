❔ 使用帮助

系统概览

SEM 颗粒智能分析系统基于 YOLOv8n-seg 实例分割，自动识别显微图像中的颗粒，并统计数量、直径分布与形态参数。

推荐总流程
  1. 1
    登录系统
    使用账号登录后进入主工作台。
  2. 2
    加载模型
    右侧「模型设置」默认选中 exp_10，也可切换其他实验模型后点击加载。
  3. 3
    上传图像
    支持单张上传或批量加载 PNG / JPG / TIF / BMP。
  4. 4
    确认比例尺
    手动输入 nm/px，或使用 OCR 识别后确认。
  5. 5
    设置过滤参数
    调整完整度、面积与边缘规则，点击「确认参数」。
  6. 6
    开始分析
    单张直接分析；多张时进入批量分析，可暂停。
  7. 7
    查看与导出
    查看指标、直径分布、明细表，并导出 CSV / Excel / 标注图。
顶部流程条会实时提示当前进度：上传 → 加载模型 → 比例尺 → 参数 → 分析 → 结果 → 导出。

登录与账号

  • 登录：输入用户名与密码进入系统。
  • 注册：需填写邀请码；用户名至少 3 位，密码至少 6 位。
  • 退出登录：右上角「退出」可安全退出并清空当前会话状态。
  • 右上角按钮：历史记录、设置、使用帮助、退出。历史与设置功能仍在完善中，本页为完整使用说明。

模型设置

  • 系统启动默认加载 exp_10 权重。
  • 下拉框可选择 exp_7 / exp_8 / exp_9 / exp_10 等本地训练模型。
  • 点击「加载选中模型」后，顶部状态会显示当前模型名称。
  • 「刷新」可重新扫描 runs/segment 下的可用模型。
  • 模拟模式：无真实模型时可用于界面预览，结果仅供演示，不可用于正式统计。

单张图片处理

  1. 在中央空白区点击「加载图像」，或拖拽一张图片到上传区。
  2. 图像显示后，可在工具栏切换「原图 / 标注图」，并使用缩放、适配窗口。
  3. 右侧设置比例尺并确认；再设置过滤参数并点击「确认参数」。
  4. 确认模型已加载后，点击「开始分析」。
  5. 分析完成后查看：有效颗粒数、平均直径、D50、平均圆度、直径分布图、圆度分布图、颗粒明细表。
  6. 需要时可「清除当前图片」后重新上传。
单张流程适合快速试跑参数；参数确认后再做批量，可保持过滤规则一致。

批量处理

如何批量加载
  • 上传区点击「批量加载」，一次选择多张图像。
  • 已有图片时，可用工具栏「追加图片」继续加入新文件（同名文件会跳过）。
  • 底部会出现「图片列表」缩略图，可点击切换当前查看的图片。
批量分析流程
  1. 先完成模型加载、比例尺确认、过滤参数确认（对整批生效）。
  2. 当图片数 ≥ 2 时，「开始分析」会启动批量分析。
  3. 系统按列表顺序逐张推理，并显示进度。
  4. 可点击「暂停分析」中断；再次点击可从断点继续（视当前版本实现）。
  5. 可用「清空全部图片」一次清除列表；或单独「清除当前图片」。
批量前请统一比例尺单位（nm/px 或 μm/px）与过滤参数，避免不同图片统计口径不一致。

比例尺设置

  • 手动模式：输入每像素对应的物理长度，单位可选 nm/px 或 μm/px，然后点「确认比例尺」。
  • OCR 模式：上传含标尺文字的图像后点击识别，核对结果后再「确认」或「编辑确认」。
  • 比例尺未确认时无法开始分析。
  • 分析结果中的直径单位为 nm，依赖于正确的比例尺换算。

过滤参数说明

  • 完整度：实际面积 / 理想圆面积，过滤残缺轮廓，默认 80%。
  • 边缘容差：判定触边的像素范围，默认 5 px。
  • 最小 / 最大面积：按像素面积过滤过小噪点或过大异常区域。
  • 排除边缘：开启后不统计触边颗粒，避免不完整对象拉低统计质量。

修改参数后必须点击「确认参数」；「重置默认参数」可恢复系统推荐值。

分析与结果解读

分析过程

系统依次执行：解码图像 → 读取比例尺 → 模型推理 → 颗粒过滤 → 统计结果。

结果面板
  • 有效颗粒：通过过滤的颗粒总数。
  • 平均直径 / D50：平均直径与中位直径（nm）。
  • 平均圆度：越接近 1 越接近理想圆。
  • 排除颗粒：因边缘、完整度、面积等原因被剔除的数量。
  • 直径分布直方图：有效颗粒平均直径的分布。
  • 圆度分布图：圆度区间统计。
  • 颗粒明细表：每个颗粒的等效圆直径、平均直径、圆度及状态；可排序并点击高亮。

导出结果

  • 分析完成后点击工具栏「导出」，打开导出中心。
  • 可勾选：统计汇总、颗粒明细、标注图像、分析参数。
  • 格式支持：Excel、CSV、PNG（标注图）；ZIP 打包能力仍在完善中。
  • 导出内容以直径与圆度为主。

管理员功能

仅管理员账号登录后可见右侧管理面板。

  • 用户管理：分页查看用户列表与角色。
  • 邀请码管理：生成邀请码、查看使用状态与使用者。

常见问题

  • 无法开始分析？请确认：已加载模型（或开启模拟）、已上传图像、已确认比例尺、已确认过滤参数。
  • 颗粒偏少？可适当降低完整度阈值，或关闭「排除边缘」后对比。
  • 噪点过多？提高最小面积或完整度阈值。
  • 直径数值异常？优先检查比例尺单位与数值是否正确。
  • 模型列表为空？确认后端已启动，且 runs/segment/exp_* 下存在 weights/best.pt 或 best.onnx。
